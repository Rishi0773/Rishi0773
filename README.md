### рЯІ† VYRA

> Your memory, indexed. Your past, queryable.

<table>
<tr>
<td width="60%">

**VYRA** is a personal memory assistant that captures text and voice memories, stores them as embeddings, and uses a **RAG (Retrieval-Augmented Generation)** workflow to answer questions about your past.

**Core capabilities:**

* рЯУЭ **Memory capture**: log text entries and voice memories
* рЯФН **Semantic search**: FAISS vector index for fast similarity lookup
* рЯ§Ц **RAG pipeline**: retrieves relevant memories as context for Gemini
* рЯУЕ **Timeline view**: browse memories by date
* рЯТђ **AI chat**: ask anything about your past, get sourced answers
* рЯУ± **Flutter frontend**: cross-platform mobile UI
